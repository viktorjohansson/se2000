Stereoscopic camera demo.

1. create two cameras - "cameraR" and "cameraL"
2. set cameras' viewport rectangles(0,0,0.5,1) and (0.5,0,0.5,1)
3. add StereoCamera.cs to the "cameraR" camera
4. set variables of the script:
	target - target
	Displacement - the stereoscopic effect
	stereo checkbox - stereoscopic effect on/off
	Distance, height, dumpings - variables for smotth following
5. Run, enjoy
